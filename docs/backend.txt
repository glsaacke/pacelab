TL;DR
Build now (MVP - 12 endpoints):

Auth: register, login, me
Strava: authorize, callback, sync, status
Activities: list, detail
Settings: get, update
Dashboard: summary

Build later (Phase 2 - 8 endpoints):

Activities: recent, delete, update
Stats: performance, PRs
Sync: detailed status, history
Strava: disconnect

Build much later (Phase 3+ - 18+ endpoints):

Everything else (password reset, account management, advanced features)




Complete API Endpoints for PaceLab
1. Authentication & User Management
csharp
// POST /api/auth/register
// Register new user
Request: { email, password }
Response: { user, token }

// POST /api/auth/login
// Login existing user
Request: { email, password }
Response: { user, token }

// POST /api/auth/logout (optional - mostly client-side with JWT)
// Invalidate token
Response: 204 No Content

// GET /api/auth/me
// Get current user info
Response: { id, email, createdAt, settings }

// POST /api/auth/refresh-token (optional - if using refresh tokens)
// Get new access token
Request: { refreshToken }
Response: { accessToken, refreshToken }

// POST /api/auth/forgot-password (later)
// Request password reset
Request: { email }
Response: 200 OK

// POST /api/auth/reset-password (later)
// Reset password with token
Request: { token, newPassword }
Response: 200 OK

4. Activities - List & Filter
csharp
// GET /api/activities
// Get paginated list of user's activities
Query: {
  page?: number (default 1),
  limit?: number (default 20),
  type?: "Ride" | "Run" | "all" (default all),
  sortBy?: "date" | "distance" | "pace" | "adjusted_pace" (default date),
  sortOrder?: "asc" | "desc" (default desc),
  startDate?: ISO date,
  endDate?: ISO date
}
Response: {
  data: [
    {
      id, name, type, distanceMeters, movingTimeSeconds,
      averageSpeedMps, adjustedSpeedMps, startDate,
      weather: { temperature, windSpeed, ... },
      adjustmentPct: 2.7
    }
  ],
  page: 1,
  limit: 20,
  total: 127,
  totalPages: 7
}

// GET /api/activities/recent
// Get last N activities (for dashboard)
Query: { count?: number (default 5) }
Response: [ ...activities ]

// GET /api/activities/summary
// Get summary stats
Query: { 
  period?: "7d" | "30d" | "90d" | "all" (default 30d),
  type?: "Ride" | "Run" | "all"
}
Response: {
  totalActivities: 45,
  totalDistance: 820000,  // meters
  totalTime: 72000,       // seconds
  averagePace: 5.8,       // m/s
  averageAdjustedPace: 6.1,
  totalElevation: 15000
}
